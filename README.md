## Project Overview

This assignment develops a parametric Grasshopper script that generates a multi-storey building solution based on variable inputs. The script outputs BIM elements directly into Revit using the Rhino.Inside Revit plugin.

---

## Design Rules and Constraints

| Parameter | Value |
|-----------|-------|
| Plot area | 24,000 to 60,000 m2 |
| Max building footprint | 60% of plot area |
| Setback from plot boundary | Min 2.4 m |
| Floor-to-floor height | 3.0 m to 5.0 m |
| Number of floors | 5 to 50 |

---

## Required BIM Outputs via Rhino.Inside Revit

- Levels for each floor
- Floor elements associated to correct levels
- Columns
- Facade with Glass material

---

## Data Summary Output

The script outputs:
- Total area of floors
- Total number of floors
- Total number of columns
- Total area of facade
