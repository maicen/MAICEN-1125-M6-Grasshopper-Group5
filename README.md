## 📌 Project Overview

This assignment develops a parametric Grasshopper script that generates a multi-storey building solution based on variable inputs. The script outputs BIM elements directly into Revit using the Rhino.Inside Revit plugin.

The goal is to shift from static manual modelling to dynamic parametric modelling where the building adapts its shape based on input parameters while following a defined set of rules.

---

## 📐 Design Rules & Constraints

| Parameter | Value |
|-----------|-------|
| Plot area | 24,000 – 60,000 m² |
| Max building footprint | 60% of plot area |
| Setback from plot boundary | Min. 2.4 m |
| Floor-to-floor height | 3.0 m – 5.0 m |
| Number of floors | 5 – 50 |

---

## 🏗️ Required BIM Outputs (via Rhino.Inside Revit)

- Levels for each floor
- Floor elements associated to correct levels
- Columns
- Façade (Direct Shape with Glass material)

---

## 📊 Data Summary Output

The script outputs:
- Total area of floors
- Total number of floors
- Total number of columns
- Total area of façade
